## Installation
Please check the [releases](https://github.com/opeik/runback/releases) tab for
the latest version of Runback.

## Building
*Ensure node, npm, and yarn are installed.*

Clone runback:
```
$ curl -sSf https://git.io/JvnMt | bash
```
Build runback:
```
$ yarn build
```
Start the server runback:
```
$ yarn start
```
